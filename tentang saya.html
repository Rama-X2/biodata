Tentang Saya
Profil Mahasiswa - Ade Ramadhani Putra
Saya adalah mahasiswa aktif di Program Studi Informatika, Universitas Linggabuana PGRI Sukabumi. Minat dan ketertarikan saya sangat kuat pada bidang teknologi, khususnya pada aspek rekayasa perangkat lunak (software) dan perangkat keras (hardware). Ketekunan saya dalam dunia teknologi tidak hanya terbatas pada teori kampus, tetapi saya juga aktif melakukan eksperimen dan pengembangan mandiri di luar jam kuliah.
Kemampuan dan Pengalaman Teknikal
Saya memiliki hobi dan keahlian dalam modifikasi perangkat elektronik, mulai dari software hingga hardware. Salah satu proyek modifikasi yang pernah saya lakukan adalah mengembangkan ulang charger KW untuk laptop agar memiliki sistem grounding yang lebih aman dan stabil. Saya tidak hanya memodifikasi secara fungsional, tetapi juga mendesain ulang body, sirkuit, dan menambahkan komponen-komponen penting seperti resistor, kapasitor, dan filter agar perangkat tersebut lebih efisien dan aman digunakan.

Dalam hal software dan sistem, saya sangat tertarik pada pengembangan dan tuning sistem operasi. Semua perangkat saya, baik laptop maupun smartphone, telah saya modifikasi dari tingkat OS, kernel, hingga sistem manajemen daya. Saya mengatur voltase, ampere per core, frekuensi prosesor, dan TDP secara manual untuk mencapai performa tinggi dengan temperatur yang tetap terjaga. Bidang ini menjadi fokus utama saya karena saya percaya bahwa performa optimal bisa dicapai tanpa harus mengorbankan efisiensi energi dan umur perangkat.
Kegiatan Profesional & Sampingan
Di luar aktivitas perkuliahan, saya juga membuka usaha jasa servis perangkat elektronik. Walaupun saya menangani berbagai jenis perangkat, mayoritas pekerjaan saya berfokus pada perbaikan HP, laptop, dan PC. Pekerjaan ini memberi saya pengalaman langsung terhadap berbagai jenis kerusakan hardware maupun software, serta keterampilan troubleshooting yang sangat terasah.

Selain itu, saya juga mengelola server RTRW-Net pribadi, yang saya bangun dan pelihara sendiri. Server ini saya gunakan untuk berbagai eksperimen jaringan, manajemen resource, dan kadang juga untuk membantu kebutuhan klien lokal.
Minat Lain dan Hobi Teknologi
Waktu luang saya sering diisi dengan berbagai aktivitas teknikal, seperti overclocking, membuat sistem pendingin kustom, dan melakukan analisis mendalam terhadap chipset dan arsitektur perangkat. Saya suka memahami detail spesifikasi teknis dan membandingkan performa antar model atau generasi komponen.

Walaupun tidak terlalu sering, saya juga menyempatkan diri bermain game, biasanya hanya saat sedang libur kerja atau tidak ada proyek teknikal yang sedang berjalan. Ketika bermain, saya juga menerapkan tuning sistem agar game berjalan dengan FPS tinggi namun tetap efisien secara termal.
